### Overview

Package to perform various installation automations.
Created by using:

- typescript
- google zx

### Install

```bash
sudo npm i -g zx

mkdir tmp
cd tmp
npm init -y

sudo npm i @aibulat/installer
```

### Uninstall

```bash
# just remove the folder where you have run npm init previosly
# for example:
rm -rf tmp
```

### Use

On Ubuntu Linux:

```bash
sudo npx i-ubuntu-mysql
npx c-vim
```
